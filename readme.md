# My Yazi dotfiles 📁

This repository contains my system configuration files, heavily customized for a keyboard-driven workflow. It currently features a tailored setup for [Yazi](https://yazi-rs.github.io/), integrated with Neovim, mpv, OnlyOffice, and Thunar.

The configs are managed using a custom installation script that safely creates symbolic links (symlinks) from this repository to the system's `~/.config` directory.

## 🚀 Installation

To deploy these dotfiles on a new machine, simply clone this repository into your home directory and run the installer.

**1. Clone the repository:**
```bash
git clone [https://github.com/](https://github.com/)<your-username>/dotfiles.git ~/dotfiles
cd ~/dotfiles
```

**2. Install dotfiles**
```bash
chmod +x install.sh
./install.sh
```
